HTML Guide

One important thing to keep in mind is that during the entire development, there will be many unknown tags and other things as such. Go to https://html.com/cheat-sheet/ to see an interactive explanation of all these tags.

Keep in mind that not all the tags will be used; you can feel free to explore those tags, but there is no need to know them (for this course).

Basic Tags

We'll start off with formatting tags, which is the simplest of the basics:
  • To bold an element, we can use the syntax <b>[text]</b>. For example, <b>Hello</b> becomes Hello.

    Keep in mind that <b>...</b> and <strong>...</strong> have the same function.
  • To italicize an element, we can use the syntax <i>[text]</i>. For example, <i>Hello</i> becomes Hello.

    Keep in mind that <i>...</i> and <em>...</em> have the same function.
  • To underline an element, we can use the syntax <u>[text]</u>. For example, <u>Hello</u> becomes Hello.

    Keep in mind that this is not good practice. For now, you can use <u>...</u>, but next class, we'll go over the "correct" way to do this.
  • To strikethrough an element, we can use the syntax <s>[text]</s>. For example, <s>Hello</s> becomes Hello.

    Keep in mind that this is not good practice. For now, you can use <s>...</s>, but next class, we'll go over the "correct" way to do this. In addition, keep in mind that <s>...</s> and <strike>...</strike> have the same function.

Note the spaces at the front, and the intentional newline (and yes, you can scroll). But in the rendered HTML output, that's not what it shows. The reason is because spaces, tabs, or newlines (all called whitespaces) are "eaten" up by HTML to separate different portions of text, and then are replaced with spaces.

So you might ask "how would I, then, be able to add a newline"? The answer is simple - use the <br /> tag. Now you might ask me two questions:
  1. Where's the closing tag?
  2. What's the small slash doing before the closing bracket of the tag?
We can answer both questions very simply - for any tag, <tag /> just means <tag></tag>. So we're not missing out on any opening tag.

Now you might ask about how to insert two spaces. This is slightly more challenging, but you'll run into the same issue later on if you're trying to write a < sign, and HTML ends up thinking it's a tag. The 6 most important "escape sequences" are:
  • &amp; = &
  • &gt; = >
  • &lt; = <
  • &quot; = "
  • &apos; = '
  • &nbsp; =   (non-breaking space character)
There are more (these can be seen when using Visual Studio Code), but they play minimal importance. To add 2 spaces, you can do &nbsp;&nbsp;, giving "Hello  James".

Lists

So far you've probably seen a lot of lists been used so far. This is actually really simple to do! You don't count numbers or anything. If you want a bulleted list, you start off with an <ul>...</ul> tag (ul stands for unordered list). For numbered (or ordered) lists, you'd use <ol>...</ol>.

But I haven't told you how to put items there yet! All items are surrounded by a <li>...</li> tag. For example, the following is an unordered list:
  • Item 1
  • Item 2
while by changing the "u" to an "o", we get
  1. Item 1
  2. Item 2

Interactive Elements

Going to any website, like Google, you'll see a lot of links and buttons; how can we make those? Links are simple; you can use the syntax <a href="[link to website">...</a> to make a link to a website.

href stands for "hypertext reference", and a stands for "anchor" (confusing as it is, you'll get it in a bit). To make a button, such as , you can just do <button>...</button>. This won't actually do much until we get into using JavaScript. To make a go to the top button, we can use IDs to reference the top, such as <a href="#[id]">...</a>.
